# 台词喵 DialogueCat 🎬

影视截图台词拼贴画工具 —— 将连贯的截图序列合成为一张高保真拼贴长图。

## 功能特性

- **智能识别** — 自动将第一张设为完整电影画幅，后续设为台词精简切片
- **全局统一排版** — 通过滑块一次性对齐所有字幕切片的位置与高度
- **独立图层微调** — 每张图片可单独启用独立裁剪参数，处理偏高或偏矮的图片
- **拖拽排序** — 支持图层卡片拖拽，灵活调整拼贴顺序
- **高保真导出** — 基于 Canvas 的无损拼接，支持 PNG / JPEG / WebP 格式
- **撤销 / 重做** — Ctrl+Z 撤销，Ctrl+Shift+Z 重做
- **预览缩放** — 支持预览缩放与自适应宽度

## 快速开始

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 技术栈

- [Vite](https://vitejs.dev/) — 构建工具
- [Tailwind CSS](https://tailwindcss.com/) v4 — 样式
- Vanilla JavaScript — 纯前端，无框架依赖

## 使用流程

1. 上传或拖拽纳影视截图到左侧上传区
2. 在左侧面板中调整每个图层的模式（完整画面 / 台词切片）
3. 使用全局滑块统一调整台词底部间距、切片高度、分割缝隙
4. 个别图片可勾选"独立裁剪参数"进行微调
5. 在顶部选择导出宽度和格式，点击"导出高清拼图"
6. 预览确认后保存到本地
